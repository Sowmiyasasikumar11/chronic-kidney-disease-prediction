# Chronic Kidney Disease Prediction System

This project predicts Chronic Kidney Disease (CKD) using Machine Learning and provides a web interface for users to enter patient medical data.

## Features
- Web-based user interface
- Machine learning prediction model
- CKD risk prediction from patient parameters

## Technologies Used
- Python
- Flask
- Scikit-learn
- HTML
- CSS

## Project Structure
```
templates/       -> Frontend HTML files
app.py           -> Flask backend
ckd_nb_model.pkl -> Trained ML model
requirements.txt -> Python dependencies
```

## How to Run

1. Install dependencies
```
pip install -r requirements.txt
```

2. Run the application
```
python app.py
```

3. Open browser
```
http://127.0.0.1:5000
```
